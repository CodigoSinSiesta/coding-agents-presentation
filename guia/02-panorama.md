# El Panorama de Herramientas

## Tiempo estimado
3 minutos

## Puntos clave
- Las herramientas se dividen en categorías según su filosofía y aproximación
- **IDE con agente nativo**: Cursor, Windsurf
  - Integración profunda en el editor
  - Experiencia fluida pero más lock-in
- **CLI / Terminal**: Claude Code, Codex, OpenCode
  - Mayor control y flexibilidad
  - Terminal-first, ideal para desarrolladores backend
- **Cloud / Nube**: Antigravity, Gemini CLI
  - Sin instalación local
  - Acceso desde cualquier máquina
- **OpenCode**: destacado como favorito del ponente
  - Open source, terminal-first
  - Multi-modelo, sin lock-in a ningún proveedor
- Cada herramienta tiene su filosofía: elige la que se adapte a tu flujo

## Datos relevantes
- Cursor es el más popular para frontend developers
- Claude Code es oficial de Anthropic, muy potente pero cerrado
- OpenCode es open source y soporta múltiples LLMs
- Codex de OpenAI está construido en Rust, similar a Claude Code
