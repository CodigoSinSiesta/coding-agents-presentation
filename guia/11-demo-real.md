# Flujo de Trabajo Real

## Tiempo estimado
4 minutos

## Puntos clave
- **Los 4 pasos que siempre uso**:
  1. **CLAUDE.md**: Definimos el contexto del proyecto
  2. **Prompt claro**: Instrucción específica con scope y criterio de éxito
  3. **Agente trabaja**: Lee archivos, razona, hace tool calls, genera código
  4. **Revisión crítica**: Verificamos, corremos tests, iteramos si es necesario
- **Pipeline de trabajo**:
  - Explorar → Analiza la estructura del proyecto
  - Planificar → Qué cambios necesitamos y por qué
  - Implementar → Genera código siguiendo la arquitectura
  - Verificar → Corre tests y valida
- **Demo en tiempo real**:
  - Claude Code lee CLAUDE.md
  - Analiza la estructura del proyecto
  - Genera casos de uso
  - Modifica archivos,  - Corre tests
  - Itera hasta completar
- **Esta presentación fue creada EN TIEMPO REAL usando Claude Code**

## Datos relevantes
- El flujo completo tarda tí vezmente 2-5 minutos
- La iteración es normal y esperada
- Los tests se generan como parte del proceso
- La revisión humana sigue siendo necesaria para código crítico
