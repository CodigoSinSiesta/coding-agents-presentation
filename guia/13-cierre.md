## Puntos clave
- **7 Takeaways principales**:
  1. El agente es Jarvis, tú eres Tony Stark. Dirige.
  2. Sin System Prompt sólido, sin resultados predecibles.
  3. Contexto es poder. Dale al agente lo que necesita.
  4. Revisa siempre. El código no es tuyo hasta que lo entiendes.
  5. Las alucinaciones se previenen, no se aceptan.
  6. Los buenos agentes son sobre todo software bien escrito (12-factor agents, Horthy/HumanLayer).
  7. Tu nuevo rol: harness engineering. Estructura el repo como prompt persistente (Lopopolo/OpenAI).
